# EtherWorld iOS

A modern iOS app for reading curated tech articles with offline support.

## Features
- Browse articles in a beautiful feed
- Read full articles with rich formatting
- Save articles for offline reading
- Dark mode support
- Responsive design

## Requirements
- iOS 16.0+
- Xcode 14.0+
- Swift 5.9+

## Building
1. Open the project in Xcode
2. Select a simulator or device
3. Press ⌘R to build and run

## Testing
- Run tests with ⌘U
- SwiftUI previews available in Canvas (⌘⌥↩︎)

## Architecture
- MVVM: Models, Views, ViewModels for clean separation
- SwiftUI: Modern declarative UI framework
- Services: Pluggable architecture for mock/real data

## Future Enhancements
- Offline storage with FileManager
- Search and filtering
- User authentication
- Notifications
